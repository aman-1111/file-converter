# React File Converter

A web app built with **React + Vite** that allows users to convert files between PDF, Word, ODT, and more.

## Features
- Convert PDF ↔ Word
- Convert ODT ↔ PDF / Word
- User-friendly interface with file upload & download
- Responsive design

## Tech Stack
- React
- Vite
- JavaScript
- HTML / CSS / Bootstrap

## How to Run Locally
1. Clone the repository:

git clone https://github.com/aman-1111/file-converter

2. Install dependencies:

npm install

3. Run the development server:

npm run dev

4. Build for production:

npm run build
